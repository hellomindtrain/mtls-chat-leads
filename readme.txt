A lightweight and essential chat button for your WordPress site.

== Description ==

MTLS Chat Leads provides a fast and clean way to connect with your customers via messaging. No bloated code, no complex settings—just a simple button that works.

**Features:**
* Floating/Sticky Chat Button.
* Shortcode support for inline buttons [mtls_chat_button].
* Customizable button colors and labels.
* Mobile-friendly design.
* Lightweight & Fast.

For advanced features like Page-wise lead tracking, date filters, and detailed analytics, consider upgrading to our Pro Version.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/mtls-chat-leads` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'MTLS Chat Leads' menu to set your number.
